# Seabream-Dogfish Analysis

## Ecological and Bioeconomic Model of a Marine Predator-Prey System


## 📖 Overview

This repository contains the complete source code for the paper:

> **"Complete Ecological and Bioeconomic Analysis of a Marine Predator-Prey System: Fear Effects, Prey Refuge, and Alternative Resources"**

The study analyzes the predator-prey interaction between:
- **Blackspot Seabream** (*Pagellus bogaraveo*) - Prey
- **Portuguese Dogfish** (*Centroscymnus coelolepis*) - Predator

in Northeast Atlantic waters (Moroccan Atlantic coast).

## 🔬 Key Features

| Feature | Parameter | Description |
|---------|-----------|-------------|
| Fear Effects | `f` | Non-consumptive predator impacts |
| Prey Refuge | `m` | Habitat protection |
| Alternative Resources | `β`, `Kp` | Generalist predator diet |
| KKT Optimization | - | Optimal harvesting strategies |
